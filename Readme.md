### Ubuntu Install
1. Use wget to download the debian package (apt-get doesn't store it)
	```
	wget https://github.com/atom/atom/releases/download/v1.18.0/atom-amd64.deb
	```
2. Unpackage the download
	```
	sudo dpkg -i atom-amd64.deb
	```

3. Install the package
	```
	sudo apt-get install -f
	```

4. You should now be able to search for Atom in your Ubuntu search
